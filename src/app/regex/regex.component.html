Regex

Check out the project on github!

I also started this project due to my love of languages. I love the FParsec parser combinator libray for F# so I decided to make my own in C# and I applied it to regular expressions so I could gain a better understanding of how they function.

This regex engine supports the common operators like +, *, ?, {{'{m, n}'}}, |, [], etc.

Currently it does not support capture groups, but this is something I hope to add to this project in the future.
