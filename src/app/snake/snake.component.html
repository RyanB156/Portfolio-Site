I originally wrote this game in JavaScript, but I ported it to TypeScript to put on this website. It uses the base array type to store a list of ordered pairs for the body of the snake. Every update cycle, the head of the snake grows in the direction of motion and the tail shrinks unless the snake ate an apple.

The snake is able to loop around the edges of the screen and will die if it collides with its tail.
